# Astute Injector - Online Update Repository

Repository ini berisi file-file konfigurasi dan skin untuk aplikasi **Astute Injector**.

## 📁 Daftar File

### 🔧 Konfigurasi
- `config_online.json` - Daftar skin dan link download
- `localconfig.json` - Konfigurasi Free Fire MAX

### 🎮 Mobile Legends Skin
- `yss_skin_cc.zip` - YSS DEFAULT TO CC
- `lancelot_skin_dawning.zip` - LANCELOT DEFAULT TO DAWNING
- `fanny_skin_aspirant.zip` - FANNY DEFAULT TO ASPIRANT
- `granger_skin_legend.zip` - GRANGER DEFAULT TO LEGEND

## 📱 Cara Menggunakan

1. Install aplikasi Astute Injector
2. Buka aplikasi, akan otomatis load daftar skin dari repository ini
3. Pilih skin yang ingin di-inject
4. Klik tombol inject dan tunggu proses selesai

## ⚠️ Peringatan

- HP harus sudah **ROOT**
- Berikan izin **All Files Access** di Android 11+
- Gunakan dengan risiko sendiri
